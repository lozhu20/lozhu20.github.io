题目描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1：

示例 2：

提示：

  • 1 <= strs.length <= 200
  • 0 <= strs[i].length <= 200
  • strs[i] 如果非空，则仅由小写英文字母组成

思路

循环比较

依次取出字符串与当前公共前缀进行比对，如果前缀都匹配，则公共前缀往后拼接一个字符，再次循环比对，重复此过程直到不匹配或公共前缀长度等于最小字符串的长度。

更好的解法

采用递推的方式，先求 strs[0] 与 strs[1] 的最长公共前缀，再求 strs[1] 与 strs[2] 的最长公共前缀。
