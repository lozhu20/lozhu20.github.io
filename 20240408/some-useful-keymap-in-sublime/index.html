sublime常用快捷键操作

zhu
2024-04-08
工具软件
sublime, 快捷键
234 words , 1 mins.

平时虽然用 sublime 比较多，但只记住文本选择的几个实用快捷键，还是在 Windows 平台上。这两天用 sublime 改代码，发现很多操作的快捷键都要现查，特意整理一下。

快捷键 操作
⌘ + T 查询指定文件
⌘ + W 关闭当前文件
⌘ + ⇧ + T 重新打开最新关闭的文件
⌘ + 数字 切换 tab 页
⌘ + L 选中当前行
⌘ + D 选中下一个相同的词
⌘ + ⌃ + g 选中当前文件中所有相同的词
⌃ + ⇧ + K 删除当前行
⌘ + K + U 转为大写
⌘ + K + L 转为小写
⌃ + G 转到指定行
⌘ + ⇧ + ↩ 在当前行前插入新行
⌘ + ↩ 在当前行后插入新行
⌘ + K + K 删除当前行光标之后的内容
⌘ + K + ⌫ 删除当前行光标之前的内容
⌘ + ⇧ + D 复制当前行
⌘ + J 合并多行
⌘ + ⌥ + [1 2 3 4] 按列拆分窗口
⌘ + ⌥ + 5 组成网格窗口
⌘ + ⌃ + 方向键 上下移动当前行
